# AI Image Similarity Search Engine

## Overview
This project finds visually similar images using deep learning feature extraction.

## Features
- Feature extraction using ResNet50
- Cosine similarity search
- Top-K similar images
- Streamlit web interface
- Feature database using pickle

## Tech Stack
- Python
- TensorFlow
- OpenCV
- Scikit-learn
- Streamlit

## Run

### Extract Features
python main.py

### Run Streamlit App
streamlit run app.py
